1. Oscillatore Armonico Smorzato

Teoria

L'oscillatore armonico smorzato descrive il moto di un sistema massa-molla soggetto a una forza di smorzamento proporzionale alla velocità. Questo modello è fondamentale per comprendere numerosi fenomeni fisici, dalle sospensioni dei veicoli ai circuiti elettrici RLC.

Equazione del Moto

L'equazione differenziale che governa il sistema è:

mẍ = -kx - vẋ

dove:

• m: massa del sistema (kg)

• k: costante elastica della molla (N/m)

• v: coefficiente di smorzamento (N·s/m)

• x: posizione rispetto all'equilibrio (m)

• ẋ = dx/dt: velocità (m/s)

• ẍ = d²x/dt²: accelerazione (m/s²)

• -kx: forza elastica di richiamo (legge di Hooke)

• -vẋ: forza di smorzamento viscoso

Dividendo per la massa m, otteniamo la forma normalizzata:

ẍ = -(k/m)x - (v/m)ẋ

Equazione Caratteristica

Per risolvere l'equazione differenziale, assumiamo una soluzione del tipo x(t) = eλt e sostituiamo nell'equazione omogenea:

λ² + (v/m)λ + (k/m) = 0

Parametri Fondamentali

Definiamo due parametri che caratterizzano completamente il comportamento del sistema:

ω = √(k/m) → Frequenza naturale (pulsazione propria)
μ = v/(2m) → Coefficiente di smorzamento normalizzato

La frequenza naturale ω rappresenta la pulsazione con cui il sistema oscillerebbe in assenza di smorzamento. Il coefficiente μ quantifica l'intensità dello smorzamento.

Sostituendo questi parametri nell'equazione caratteristica:

λ² + 2μλ + ω² = 0

Le soluzioni sono:

λ₁,₂ = -μ ± √(μ² - ω²)

Analisi dei Tre Casi

Il comportamento del sistema dipende dal segno del discriminante Δ = μ² - ω², che determina la natura delle radici λ₁ e λ₂.

Caso 1: Sovrasmorzamento (μ > ω)

Quando il coefficiente di smorzamento è maggiore della frequenza naturale, abbiamo:

Δ = μ² - ω² > 0 → Radici reali e distinte

Le due radici sono:

λ₁ = -μ + √(μ² - ω²) (radice meno negativa)
λ₂ = -μ - √(μ² - ω²) (radice più negativa)

Entrambe le radici sono negative (λ₁ < 0 e λ₂ < 0), quindi la soluzione generale è:

x(t) = C₁eλ₁t + C₂eλ₂t

Comportamento fisico: Il sistema non oscilla. Ritorna all'equilibrio in modo esponenziale e lento. Maggiore è μ rispetto a ω, più lento è il ritorno all'equilibrio. È il caso delle sospensioni "troppo rigide".

Caso 2: Smorzamento Critico (μ = ω)

Quando lo smorzamento è esattamente uguale alla frequenza naturale:

Δ = μ² - ω² = 0 → Radice doppia

La radice doppia è:

λ = -μ

Per radici doppie, la soluzione generale assume la forma:

x(t) = (C₁ + C₂t)e-μt

Comportamento fisico: Il sistema non oscilla e ritorna all'equilibrio nel minor tempo possibile senza oscillare. Questo è il regime ottimale per molte applicazioni ingegneristiche (porte automatiche, sospensioni ottimizzate). Rappresenta il confine tra sovrasmorzamento e sottosmorzamento.

Caso 3: Sottosmorzamento (μ < ω)

Quando lo smorzamento è minore della frequenza naturale:

Δ = μ² - ω² < 0 → Radici complesse coniugate

Le radici complesse sono:

λ = -μ ± iωd

dove la frequenza smorzata è:

ωd = √(ω² - μ²)

La soluzione generale è:

x(t) = e-μt[C₁cos(ωdt) + C₂sin(ωdt)]

Comportamento fisico: Il sistema oscilla attorno alla posizione di equilibrio con frequenza ωd, ma l'ampiezza delle oscillazioni decresce esponenzialmente secondo il fattore e-μt. Maggiore è μ, più rapidamente si smorzano le oscillazioni. È il caso tipico delle oscillazioni reali.

NOTA: La frequenza smorzata ωd è sempre minore della frequenza naturale ω. Quando μ → 0, ωd → ω (oscillatore non smorzato).

Esempio Svolto

Problema

Risolvere il sistema:

ẍ + 6ẋ + 5x = 0
Condizioni iniziali: x(0) = 1, ẋ(0) = 0

Soluzione

Passo 1: Scriviamo l'equazione caratteristica

λ² + 6λ + 5 = 0

Passo 2: Identifichiamo i parametri

Confrontando con λ² + 2μλ + ω² = 0:

2μ = 6 → μ = 3
ω² = 5 → ω = √5 ≈ 2.236

Passo 3: Determiniamo il tipo di smorzamento

Poiché μ = 3 > ω = √5 ≈ 2.236, siamo nel caso di sovrasmorzamento.

Passo 4: Calcoliamo le radici

Δ = μ² - ω² = 9 - 5 = 4 > 0

λ₁ = -μ + √(μ² - ω²) = -3 + √4 = -3 + 2 = -1
λ₂ = -μ - √(μ² - ω²) = -3 - √4 = -3 - 2 = -5

Passo 5: Scriviamo la soluzione generale

x(t) = C₁e-t + C₂e-5t

Passo 6: Calcoliamo la derivata

ẋ(t) = -C₁e-t - 5C₂e-5t

Passo 7: Applichiamo le condizioni iniziali

Per t = 0:

x(0) = C₁e0 + C₂e0 = C₁ + C₂ = 1
ẋ(0) = -C₁e0 - 5C₂e0 = -C₁ - 5C₂ = 0

Sistema:

C₁ + C₂ = 1
-C₁ - 5C₂ = 0

Dalla seconda equazione: C₁ = -5C₂

Sostituendo nella prima: -5C₂ + C₂ = 1 → -4C₂ = 1 → C₂ = -1/4

Quindi: C₁ = -5(-1/4) = 5/4

Soluzione finale:

x(t) = (5/4)e-t - (1/4)e-5t

Interpretazione fisica: Il sistema parte dalla posizione x = 1 con velocità nulla e ritorna lentamente all'equilibrio senza oscillare. Il termine e-t decade più lentamente del termine e-5t, quindi per tempi lunghi il comportamento è dominato da (5/4)e-t.

Esempio Svolto

Problema

Risolvere il sistema:

ẍ + 2ẋ + 5x = 0
Condizioni iniziali: x(0) = 1, ẋ(0) = 0

Soluzione

Passo 1: Scriviamo l'equazione caratteristica

λ² + 2λ + 5 = 0

Passo 2: Identifichiamo i parametri

Confrontando con λ² + 2μλ + ω² = 0:

2μ = 2 → μ = 1
ω² = 5 → ω = √5 ≈ 2.236

Inoltre, dall'equazione originale: m = 1, v = 2, k = 5

Passo 3: Determiniamo il tipo di smorzamento

Poiché μ = 1 < ω = √5 ≈ 2.236, siamo nel caso di sottosmorzamento.

Passo 4: Calcoliamo le radici complesse

Δ = μ² - ω² = 1 - 5 = -4 < 0

Le radici sono complesse coniugate:

λ₁,₂ = -μ ± i√(ω² - μ²) = -1 ± i√4 = -1 ± 2i

Quindi:

λ₁ = -1 + 2i
λ₂ = -1 - 2i

Passo 5: Scriviamo la soluzione generale

Per radici complesse λ = -μ ± iωd, la soluzione è:

x(t) = C₁eλ₁t + C₂eλ₂t

Con λ₁ = -1 + 2i e λ₂ = -1 - 2i:

x(t) = C₁e(-1+2i)t + C₂e(-1-2i)t

Passo 6: Calcoliamo la derivata

ẋ(t) = λ₁C₁eλ₁t + λ₂C₂eλ₂t
= (-1 + 2i)C₁e(-1+2i)t + (-1 - 2i)C₂e(-1-2i)t

Passo 7: Applichiamo le condizioni iniziali

Per t = 0:

x(0) = C₁e0 + C₂e0 = C₁ + C₂ = 1
ẋ(0) = (-1 + 2i)C₁ + (-1 - 2i)C₂ = 0

Sistema da risolvere:

C₁ + C₂ = 1
(-1 + 2i)C₁ + (-1 - 2i)C₂ = 0

Dalla seconda equazione:

(-1 + 2i)C₁ + (-1 - 2i)C₂ = 0
(-1 + 2i)C₁ = (1 + 2i)C₂
C₁ = [(1 + 2i)/(−1 + 2i)]C₂

Razionalizziamo moltiplicando per il coniugato:

C₁ = [(1 + 2i)(−1 − 2i)]/[(−1 + 2i)(−1 − 2i)]C₂
= [(−1 − 2i − 2i − 4i²)]/[1 + 4]C₂
= [(−1 − 4i + 4)]/5 C₂ = (3 − 4i)/5 C₂

Dalla prima equazione C₁ + C₂ = 1:

(3 − 4i)/5 C₂ + C₂ = 1
[(3 − 4i) + 5]/5 C₂ = 1
(8 − 4i)/5 C₂ = 1
C₂ = 5/(8 − 4i)

Razionalizziamo:

C₂ = [5(8 + 4i)]/[(8 − 4i)(8 + 4i)] = [5(8 + 4i)]/(64 + 16) = (40 + 20i)/80 = (1 + 2i)/4

E quindi:

C₁ = 1 − C₂ = 1 − (1 + 2i)/4 = (4 − 1 − 2i)/4 = (3 − 2i)/4

Ma per semplificare, notiamo un approccio alternativo. Dalla seconda equazione:

−C₁ + 2iC₁ − C₂ − 2iC₂ = 0
−(C₁ + C₂) + 2i(C₁ − C₂) = 0

Usando C₁ + C₂ = 1:

−1 + 2i(C₁ − C₂) = 0
C₁ − C₂ = 1/(2i) = −i/2

Sistema più semplice:

C₁ + C₂ = 1
C₁ − C₂ = −i/2

Sommando: 2C₁ = 1 − i/2 → C₁ = (2 − i)/4

Sottraendo: 2C₂ = 1 + i/2 → C₂ = (2 + i)/4

Soluzione finale:

x(t) = [(2 − i)/4]e(-1+2i)t + [(2 + i)/4]e(-1-2i)t

Questa può essere scritta in forma reale usando le formule di Eulero, ottenendo:

x(t) = e-t[A cos(2t) + B sin(2t)]

Interpretazione fisica: Il sistema oscilla con frequenza smorzata ωd = 2 rad/s, mentre l'ampiezza decresce esponenzialmente con costante di tempo 1/μ = 1 secondo.

2. Oscillatore Forzato

Teoria

L'oscillatore forzato è un oscillatore armonico smorzato sottoposto a una forza esterna periodica. Questo modello descrive fenomeni importanti come la risonanza meccanica ed elettrica.

Equazione del Moto

L'equazione differenziale è:

mẍ + vẋ + kx = F cos(γt)

dove:

• F: ampiezza massima della forza forzante (N)

• γ: frequenza angolare della forza esterna (rad/s)

• Gli altri parametri sono gli stessi dell'oscillatore smorzato

Parametri Normalizzati

Dividendo per m:

ω² = k/m (frequenza naturale al quadrato)
μ = v/(2m) (coefficiente di smorzamento)
f = F/m (forza forzante per unità di massa)

L'equazione diventa:

ẍ + 2μẋ + ω²x = f cos(γt)

Struttura della Soluzione

La soluzione generale di un'equazione differenziale non omogenea è data dalla somma di due contributi:

x(t) = xom(t) + xp(t)

dove:

• xom(t): soluzione dell'equazione omogenea (transitorio)

• xp(t): soluzione particolare (regime stazionario)

Soluzione Omogenea (Transitorio)

La soluzione omogenea è identica a quella dell'oscillatore smorzato e dipende dal rapporto tra μ e ω:

Caso μ < ω (Sottosmorzamento)

xom(t) = e-μt[C₁cos(ωdt) + C₂sin(ωdt)]
dove ωd = √(ω² - μ²)

Caso μ = ω (Smorzamento Critico)

xom(t) = (C₁ + C₂t)e-μt

Caso μ > ω (Sovrasmorzamento)

xom(t) = C₁eλ₁t + C₂eλ₂t
dove λ₁,₂ = -μ ± √(μ² - ω²)
IMPORTANTE: Il termine omogeneo xom(t) tende a zero per t → ∞ a causa del fattore e-μt. Per questo viene chiamato "transitorio". Dopo un tempo sufficientemente lungo, rimane solo la soluzione particolare.

Soluzione Particolare (Regime Stazionario)

Per trovare la soluzione particolare, cerchiamo una soluzione della stessa forma della forzante:

xp(t) = a cos(γt) + b sin(γt)

Procedimento di Calcolo

Passo 1: Calcoliamo le derivate

ẋp(t) = -aγ sin(γt) + bγ cos(γt)
ẍp(t) = -aγ² cos(γt) - bγ² sin(γt)

Passo 2: Sostituiamo nell'equazione differenziale

[-aγ² cos(γt) - bγ² sin(γt)] + 2μ[-aγ sin(γt) + bγ cos(γt)] + ω²[a cos(γt) + b sin(γt)] = f cos(γt)

Passo 3: Raggruppiamo i termini in cos(γt) e sin(γt)

Per cos(γt):

-aγ² + 2μbγ + ω²a = f
a(ω² - γ²) + 2μbγ = f

Per sin(γt):

-bγ² - 2μaγ + ω²b = 0
b(ω² - γ²) - 2μaγ = 0

Passo 4: Risolviamo il sistema lineare

a(ω² - γ²) + 2μbγ = f
-2μaγ + b(ω² - γ²) = 0

Dalla seconda equazione: b = (2μaγ)/(ω² - γ²)

Sostituendo nella prima e risolvendo per a:

a = [f(ω² - γ²)] / [(ω² - γ²)² + (2μγ)²]
b = [f(2μγ)] / [(ω² - γ²)² + (2μγ)²]

Forma Compatta

La soluzione particolare può essere scritta in forma compatta come:

xp(t) = A cos(γt - φ)

dove:

A = f / √[(ω² - γ²)² + (2μγ)²] (ampiezza)
tan(φ) = (2μγ)/(ω² - γ²) (sfasamento)

Fenomeno della Risonanza

L'ampiezza A della risposta stazionaria dipende dalla frequenza forzante γ. Studiando A(γ), si trova che essa raggiunge un massimo per:

γris = √(ω² - 2μ²) (frequenza di risonanza)

L'ampiezza massima è:

Amax = f / (2μ√(ω² - μ²))
RISONANZA: Quando γ ≈ ω (frequenza forzante vicina alla frequenza naturale) e μ è piccolo, l'ampiezza diventa molto grande. Questo può causare danni strutturali (ponti, edifici) se non controllato. Per questo motivo le truppe militari rompono il passo quando attraversano i ponti.

Soluzione Completa

La soluzione completa del problema è:

x(t) = xom(t) + xp(t)

Le costanti C₁ e C₂ nella soluzione omogenea si determinano imponendo le condizioni iniziali x(0) e ẋ(0) sulla soluzione completa.

Comportamento Asintotico

Per tempi lunghi (t → ∞):

• Il transitorio xom(t) → 0 a causa dello smorzamento

• Rimane solo il regime stazionario xp(t)

• Il sistema oscilla con la frequenza della forzante γ

• L'ampiezza e lo sfasamento dipendono da γ, ω e μ

3. Baricentro e Momento d'Inerzia

Teoria

Il baricentro (o centro di massa) è il punto in cui si può considerare concentrata tutta la massa di un corpo. Il momento d'inerzia misura la resistenza di un corpo alla rotazione attorno a un asse.

Definizioni Fondamentali

Baricentro:

XG = (1/m) ∫∫ x ρ(x,y) dA
YG = (1/m) ∫∫ y ρ(x,y) dA

Momento d'Inerzia:

I = ∫∫ r² ρ(x,y) dA

dove r è la distanza dall'asse di rotazione.

Teorema di Huygens-Steiner

Questo teorema fondamentale permette di calcolare il momento d'inerzia rispetto a un asse qualsiasi conoscendo quello rispetto al baricentro:

IA = IG + md²

dove:

• IA: momento d'inerzia rispetto all'asse A

• IG: momento d'inerzia rispetto al baricentro

• d: distanza tra i due assi paralleli

Asta Omogenea

Massa Totale

Per un'asta di lunghezza L con densità lineare costante ρ:

m = ρL

Baricentro

Ponendo un'estremità nell'origine:

XG = (1/m) ∫0L x ρ dx = (ρ/m) ∫0L x dx

Sostituendo ρ = m/L:

XG = (m/L)(1/m)[x²/2]0L = (1/L)(L²/2) = L/2

Il baricentro si trova a metà dell'asta, come intuitivamente ci si aspetta per un corpo omogeneo.

Momento d'Inerzia rispetto a un'Estremità

IA = ∫0L x² ρ dx = ρ[x³/3]0L = ρL³/3

Sostituendo ρ = m/L:

IA = (m/L)(L³/3) = mL²/3

Momento d'Inerzia rispetto al Baricentro

Applicando il teorema di Steiner con d = L/2:

IG = IA - md² = (mL²/3) - m(L/2)² = (mL²/3) - (mL²/4) = mL²/12

Quadrato Omogeneo

Consideriamo un quadrato di lato L con densità superficiale ρ costante.

Massa Totale

m = ρA = ρL²

Baricentro

XG = (1/m) ∫0L∫0L x ρ dx dy = (ρ/m) ∫0L x dx ∫0L dy = (ρ/m)(L²/2)L = L/2
YG = L/2

Il baricentro coincide con il centro geometrico.

Momento d'Inerzia rispetto a un Angolo

IA = ∫0L∫0L (x² + y²) ρ dx dy = ρ[∫0L x² dx ∫0L dy + ∫0L dx ∫0L y² dy]
= ρ[(L³/3)L + L(L³/3)] = (2ρL⁴/3) = (2/3)mL²

Momento d'Inerzia rispetto al Centro

La distanza dal centro a un angolo è d = L√2/2:

IG = IA - md² = (2/3)mL² - m(L√2/2)² = (2/3)mL² - (mL²/2) = (mL²/6)

Rettangolo Omogeneo

Rettangolo con base L e altezza H, densità ρ costante.

Massa Totale

m = ρLH

Baricentro

XG = (1/m) ∫0L∫0H x ρ dx dy = L/2
YG = H/2

Momenti d'Inerzia

Rispetto all'asse x (orizzontale passante per la base):

IxA = ∫0L∫0H y² ρ dx dy = ρL(H³/3) = (m/3)H²

Rispetto all'asse y (verticale passante per un lato):

IyA = ∫0L∫0H x² ρ dx dy = ρH(L³/3) = (m/3)L²

Rispetto al baricentro:

IxG = IxA - m(H/2)² = (m/3)H² - (m/4)H² = (m/12)H²
IyG = IyA - m(L/2)² = (m/3)L² - (m/4)L² = (m/12)L²
NOTA IMPORTANTE: Per Ix integriamo y² (non x²) perché il momento d'inerzia rispetto all'asse x misura la distribuzione della massa in direzione perpendicolare all'asse x, cioè lungo y. La distanza di un punto (x,y) dall'asse x è semplicemente |y|.

Disco Omogeneo

Disco di raggio R con densità superficiale ρ costante.

Massa Totale

m = ρπR²

Baricentro

Per simmetria, il baricentro coincide con il centro:

XG = 0, YG = 0

Momento d'Inerzia rispetto al Centro

Usando coordinate polari (x = r cos θ, y = r sin θ, dA = r dr dθ):

IO = ∫02π∫0R r² ρ r dr dθ = ρ ∫02π dθ ∫0R r³ dr
= ρ(2π)(R⁴/4) = (πρR⁴/2) = (m/2)R²

Momento d'Inerzia rispetto a un Asse Diametrale

Per simmetria, il momento d'inerzia rispetto a qualsiasi diametro è lo stesso. Calcolando rispetto all'asse x:

Ix = ∫02π∫0R (r sin θ)² ρ r dr dθ = ρ ∫02π sin² θ dθ ∫0R r³ dr

Usando l'integrale noto ∫02π sin² θ dθ = π:

Ix = ρπ(R⁴/4) = (m/4)R²

Semidisco Omogeneo

Metà superiore di un disco di raggio R.

Massa Totale

m = (ρπR²)/2

Baricentro

Per simmetria: XG = 0

Calcoliamo YG:

YG = (1/m) ∫0π∫0R (r sin θ) ρ r dr dθ
= (2/πρR²) ρ ∫0π sin θ dθ ∫0R r² dr = (2/πR²)[-cos θ]0π(R³/3)
= (2/πR²)(2)(R³/3) = (4R)/(3π)

Momenti d'Inerzia

Rispetto al centro (asse perpendicolare al piano):

IO = ∫0π∫0R r² ρ r dr dθ = ρπ(R⁴/4) = (m/2)R²

Rispetto alla base (asse orizzontale):

IB = ρ ∫0π sin² θ dθ ∫0R r³ dr

Usando ∫0π sin² θ dθ = π/2:

IB = ρ(π/2)(R⁴/4) = (m/4)R²

Corona Circolare Omogenea

Anello circolare con raggio interno R₁ e raggio esterno R₂.

Massa Totale

m = ρπ(R₂² - R₁²)

Baricentro

Per simmetria: XG = 0, YG = 0

Momento d'Inerzia rispetto al Centro

IO = ∫02π∫R₁R₂ r² ρ r dr dθ = ρ(2π)[r⁴/4]R₁R₂
= (ρπ/2)(R₂⁴ - R₁⁴) = (m/2)(R₂² + R₁²)

4. Energia Cinetica

Teoria - Teorema di König

Per calcolare l'energia cinetica di un corpo rigido che si muove nel piano, utilizziamo il Teorema di König, che permette di scomporre l'energia cinetica in due contributi:

T = Ttrasl + Trot

dove:

• Ttrasl: energia cinetica traslazionale del baricentro

• Trot: energia cinetica rotazionale attorno al baricentro

Forma Generale

L'energia cinetica rispetto a un polo O generico è:

T = (1/2)mVO² + m(OG) × [VO × ω] + (1/2)ω · I(ω)

dove:

• Primo termine: energia cinetica del polo O

• Secondo termine: termine misto (si annulla se O ≡ G)

• Terzo termine: energia cinetica rotazionale

Casi Particolari

Caso 1: Polo nel Baricentro (O ≡ G)

Il termine misto scompare:

T = (1/2)mVG² + (1/2)IGω²

Caso 2: Punto Fisso O

Se il corpo ha un punto fisso O (VO = 0):

T = (1/2)IOω²

Solo energia rotazionale attorno al punto fisso.

Caso 3: Rototraslazione Piana

Corpo che scorre su una superficie piana:

T = (1/2)mVG² + (1/2)IGω²

Caso 4: Rotazione Pura attorno a un Punto A

Un estremo del corpo è incernierato al punto A:

T = (1/2)IAω²

Esempio Svolto: Lamina Rettangolare

Problema

Una lamina rettangolare omogenea di lati a e b ha l'estremo A vincolato a scorrere su una retta orizzontale. Calcolare l'energia cinetica per:

a) Densità ρ costante

Soluzione

Passo 1: Calcolo del Baricentro

Ponendo A nell'origine con il rettangolo allineato agli assi:

XG = a/2, YG = b/2

Passo 2: Posizione del Baricentro dopo Rotazione

Se il rettangolo ruota di un angolo θ attorno ad A, il baricentro si muove. Usiamo la matrice di rotazione R(θ):

            R(θ) = |cos θ  -sin θ|
                   |sin θ   cos θ|

Il vettore posizione del baricentro ruotato:

            X'G = R(θ) · XG = |cos θ  -sin θ| |a/2|
                              |sin θ   cos θ| |b/2|

            = |(a/2)cos θ - (b/2)sin θ|
              |(a/2)sin θ + (b/2)cos θ|

Ma A può anche traslare lungo x di una quantità ẋ, quindi:

X'G = x + (a/2)cos θ - (b/2)sin θ
Y'G = (a/2)sin θ + (b/2)cos θ

Passo 3: Velocità del Baricentro

Deriviamo rispetto al tempo:

VGx = ẋ - (a/2)θ̇ sin θ - (b/2)θ̇ cos θ
VGy = (a/2)θ̇ cos θ - (b/2)θ̇ sin θ

Passo 4: Modulo Quadro della Velocità

VG² = VGx² + VGy²

Sviluppando i quadrati:

VGx² = ẋ² - ẋθ̇(a sin θ + b cos θ) + (θ̇²/4)(a² sin² θ + b² cos² θ + 2ab sin θ cos θ)
VGy² = (θ̇²/4)(a² cos² θ + b² sin² θ - 2ab sin θ cos θ)

Sommando:

VG² = ẋ² - ẋθ̇(a sin θ + b cos θ) + (θ̇²/4)(a² + b²)

Passo 5: Momento d'Inerzia

Massa: m = ρab, quindi ρ = m/(ab)

Momento d'inerzia rispetto ad A:

IA = ρ ∫0a∫0b (x² + y²) dx dy = ρ[(a³b/3) + (ab³/3)]
= (ρab/3)(a² + b²) = (m/3)(a² + b²)

Alternativamente, usando Steiner con d = √(a² + b²)/2:

IG = IA - md² = (m/3)(a² + b²) - m(a² + b²)/4 = (m/12)(a² + b²)

Passo 6: Energia Cinetica Totale

T = (1/2)mVG² + (1/2)IGω²
T = (1/2)m[ẋ² - ẋθ̇(a sin θ + b cos θ) + (θ̇²/4)(a² + b²)] + (1/2) · (m/12)(a² + b²) · θ̇²
T = (m/2)ẋ² - (m/2)ẋθ̇(a sin θ + b cos θ) + (m/8)θ̇²(a² + b²) + (m/24)θ̇²(a² + b²)
T = (m/2)ẋ² - (m/2)ẋθ̇(a sin θ + b cos θ) + (m/6)θ̇²(a² + b²)
NOTA: Per il caso b) con densità variabile ρ(x,y), si devono ricalcolare massa, baricentro e momenti d'inerzia seguendo gli stessi passaggi ma tenendo conto della funzione ρ(x,y) specifica.

5. Tensori d'Inerzia

Teoria

Il tensore d'inerzia è una matrice 3×3 simmetrica che descrive completamente come la massa di un corpo rigido è distribuita rispetto a un sistema di riferimento tridimensionale. È fondamentale per lo studio della dinamica rotazionale nello spazio.

Forma Generale del Tensore

	    |  Ixx  -Ixy  -Ixz |
	I = | -Iyx   Iyy  -Iyz |
	    | -Izx  -Izy   Izz |

Momenti d'Inerzia Principali

Gli elementi diagonali sono i momenti d'inerzia rispetto agli assi coordinati:

Ixx = ∫V ρ(y² + z²) dV
Iyy = ∫V ρ(x² + z²) dV
Izz = ∫V ρ(x² + y²) dV

Ogni momento d'inerzia misura la resistenza alla rotazione attorno al rispettivo asse. Ad esempio, Ixx misura quanto la massa è distribuita lontano dall'asse x.

Prodotti d'Inerzia

Gli elementi fuori diagonale sono i prodotti d'inerzia:

Ixy = Iyx = ∫V ρxy dV
Ixz = Izx = ∫V ρxz dV
Iyz = Izy = ∫V ρyz dV

I prodotti d'inerzia sono nulli quando:

• Il sistema di riferimento è centrato nel baricentro

• Gli assi sono allineati con gli assi di simmetria del corpo

Casi Bidimensionali (z = 0)

Per figure piane nel piano xy, l'integrazione si riduce a integrali doppi sull'area:

Ixx = ∫∫ ρy² dA
Iyy = ∫∫ ρx² dA
Izz = ∫∫ ρ(x² + y²) dA = Ixx + Iyy

Coordinate Polari

Per corpi con simmetria circolare (disco, semidisco, corona), usiamo coordinate polari:

x = r cos θ, y = r sin θ, dA = r dr dθ

Ixx = ∫∫ ρr² sin² θ · r dr dθ
Iyy = ∫∫ ρr² cos² θ · r dr dθ
Izz = ∫∫ ρr²(sin² θ + cos² θ) · r dr dθ = ∫∫ ρr³ dr dθ

Proprietà del Tensore

1. Simmetria: Iij = Iji

2. Assi principali: Esistono tre assi ortogonali (assi principali d'inerzia) per cui il tensore diventa diagonale

3. Invarianti: La traccia del tensore (Ixx + Iyy + Izz) è invariante rispetto a rotazioni

Esempi di Calcolo

Disco Omogeneo

Per un disco di raggio R nel piano xy:

Ixx = ∫02π∫0R ρr² sin² θ · r dr dθ = (m/4)R²
Iyy = ∫02π∫0R ρr² cos² θ · r dr dθ = (m/4)R²
Izz = ∫02π∫0R ρr³ dr dθ = (m/2)R²

Si verifica che Izz = Ixx + Iyy, come previsto per corpi piani.

Rettangolo Omogeneo

Per un rettangolo di lati a e b centrato nell'origine:

Ixx = ∫-a/2a/2∫-b/2b/2 ρy² dx dy = (m/12)b²
Iyy = ∫-a/2a/2∫-b/2b/2 ρx² dx dy = (m/12)a²
Izz = (m/12)(a² + b²)

Tutti i prodotti d'inerzia sono nulli per simmetria.

APPLICAZIONE: Il tensore d'inerzia è essenziale nella robotica e nell'ingegneria aerospaziale per calcolare le equazioni di Eulero per la dinamica rotazionale: I·ω̇ + ω × (I·ω) = M, dove M è il momento esterno applicato.

6. Triedro di Frenet

Teoria

Il triedro di Frenet (o triedro principale) è un sistema di riferimento mobile che si adatta in ogni punto a una curva regolare nello spazio tridimensionale. È fondamentale per studiare la geometria differenziale delle curve e per descrivere il moto di un punto lungo una traiettoria.

Definizione della Curva

Data una curva regolare nello spazio tridimensionale parametrizzata rispetto all'ascissa curvilinea s:

X(s) = (x(s), y(s), z(s))

dove s è la lunghezza d'arco misurata lungo la curva a partire da un punto di riferimento. L'uso dell'ascissa curvilinea garantisce che |dX/ds| = 1.

I Tre Versori del Triedro

1. Versore Tangente (t)

Il versore tangente indica la direzione istantanea della curva:

t = dX/ds

Per costruzione, |t| = 1 (è un versore). La tangente punta nella direzione del moto lungo la curva.

2. Versore Normale (n)

Il versore normale è ortogonale al versore tangente e punta verso il "centro di curvatura" della traiettoria:

n = (1/κ) · dt/ds

dove κ (kappa) è la curvatura della curva:

κ = |dt/ds|

La curvatura misura quanto "rapidamente" la curva cambia direzione. Il suo reciproco ρ = 1/κ è il raggio di curvatura.

3. Versore Binormale (b)

Il versore binormale è ortogonale sia a t che a n, ed è definito dal prodotto vettoriale:

b = t × n

Il versore binormale indica la direzione attorno alla quale la curva si "torce" nello spazio.

Proprietà del Triedro

1. I tre versori t, n, b formano una terna ortonormale destrorsa

2. t · n = 0, t · b = 0, n · b = 0

3. |t| = |n| = |b| = 1

Piano Osculatore

Il piano contenente t e n è chiamato piano osculatore. È il piano che "meglio approssima" la curva in quel punto. La proiezione della curva sul piano osculatore mostra la curvatura locale.

Formule di Frenet-Serret

Le formule di Frenet-Serret descrivono come variano i tre versori lungo la curva:

dt/ds = κn
dn/ds = -κt + τb
db/ds = -τn

dove:

• κ (curvatura): misura quanto la curva si piega

• τ (torsione): misura quanto la curva si torce uscendo dal piano osculatore

Significato della Torsione

La torsione τ è definita da:

τ = -n · (db/ds)

• Se τ = 0: la curva è piana (rimane sempre nello stesso piano)

• Se τ ≠ 0: la curva è sghemba (tridimensionale)

Caso Particolare: Curve Piane

Per una curva contenuta in un piano (ad esempio il piano xy):

Torsione: τ = 0

Proprietà delle curve piane:

• Il piano osculatore T-N è fisso e coincide con il piano della curva

• Il versore binormale b è costante e perpendicolare al piano

• Solo la curvatura κ varia lungo la curva

• Le formule di Frenet si semplificano:

dt/ds = κn
dn/ds = -κt
db/ds = 0

Calcolo Pratico dei Versori

Il procedimento standard per calcolare il triedro di Frenet consiste nell'uso della parametrizzazione con ascissa curvilinea. Data una curva parametrizzata X(α) con un parametro generico α (che può essere un angolo θ, un tempo t, o altro):

Metodo Generale con Parametro Arbitrario

Passo 1: Versore Tangente

Il versore tangente si calcola applicando la regola della catena:

t = dX/ds = (dX/dα) · (dα/ds)

dove dα/ds si ricava dall'ascissa curvilinea:

s = ∫α₀α |dX/dα'| dα' → ds/dα = |dX/dα| → dα/ds = 1/|dX/dα|

Quindi:

t = (dX/dα) / |dX/dα|

Passo 2: Versore Normale e Raggio di Curvatura

Il versore normale è dato da:

n = ρ · (dt/ds) = ρ · (dt/dα) · (dα/ds)

dove ρ (rho) è il raggio di curvatura, definito come:

ρ = |dt/ds|⁻¹ = |dt/dα · dα/ds|⁻¹

Il raggio di curvatura è l'inverso della curvatura: ρ = 1/κ

Passo 3: Versore Binormale

Per curve piane (nel piano xy), il versore binormale si calcola con il prodotto vettoriale:

b = t × n

Calcolo pratico del prodotto vettoriale:

b = | i j k |
      | tx ty tz |
      | nx ny nz |

Componenti:

bx = tynz - tzny
by = tznx - txnz
bz = txny - tynx

Esempio 1: Circonferenza

Curva Parametrica

Circonferenza di raggio R nel piano xy:

X(θ) = (R cos θ, R sin θ)

Passo 1: Versore Tangente

Calcoliamo dX/dθ:

dX/dθ = (-R sin θ, R cos θ)

Calcoliamo l'ascissa curvilinea:

s = ∫θ₀θ |dX/dθ'| dθ' → ds/dθ = |dX/dθ| = R

Quindi:

dθ/ds = 1/R

Il versore tangente:

t = dX/ds = (dX/dθ) · (dθ/ds) = (-R sin θ, R cos θ) · (1/R) = (-sin θ, cos θ)

Passo 2: Versore Normale

Calcoliamo dt/dθ:

dt/dθ = (-cos θ, -sin θ)

Calcoliamo il raggio di curvatura:

ρ = |dt/dθ · dθ/ds|⁻¹ = |(-cos θ, -sin θ) · (1/R)|⁻¹
= |√(cos²θ + sin²θ)/R|⁻¹ = |1/R|⁻¹ = R

Il versore normale:

n = ρ · (dt/ds) = R · (dt/dθ) · (dθ/ds)
= R · (-cos θ, -sin θ) · (1/R) = (-cos θ, -sin θ)

Passo 3: Versore Binormale

Prodotto vettoriale:

b = t × n = | i j k |
                    | -sin θ cos θ 0 |
                    | -cos θ -sin θ 0 |

Componenti:

bx = cos θ · 0 - 0 · (-sin θ) = 0
by = 0 · (-cos θ) - (-sin θ) · 0 = 0
bz = (-sin θ) · (-sin θ) - cos θ · (-cos θ) = sin²θ + cos²θ = 1

Quindi:

b = (0, 0, 1)

Interpretazione: Il versore binormale è costante e perpendicolare al piano della circonferenza, confermando che si tratta di una curva piana con torsione τ = 0.

Esempio 2: Parabola

Curva Parametrica

Parabola nel piano xy:

X(α) = (α, aα²)

Passo 1: Versore Tangente

Calcoliamo dX/dα:

dX/dα = (1, 2aα)

Ascissa curvilinea:

|dX/dα| = √(1 + 4a²α²) → dα/ds = 1/√(1 + 4a²α²)

Versore tangente:

t = (1, 2aα) · 1/√(1 + 4a²α²) = (1/√(1 + 4a²α²), 2aα/√(1 + 4a²α²))

Passo 2: Versore Normale

Calcoliamo dt/dα. Ponendo f = (1 + 4a²α²) per semplicità:

dt/dα = (-4a²α · f⁻³/², 2a · f⁻¹/² - 8a³α² · f⁻³/²)

Semplificando:

dt/dα = (-4a²α · f⁻³/², 2a(1 + 4a²α² - 4a²α²) · f⁻³/²)
= (-4a²α · f⁻³/², 2a · f⁻³/²)

Raggio di curvatura:

ρ = |dt/dα · dα/ds|⁻¹ = |(-4a²α · f⁻³/², 2a · f⁻³/²) · f⁻¹/²|⁻¹
= |√(16a⁴α² · f⁻⁴ + 4a² · f⁻⁴)|⁻¹ = |√(4a²(4a²α² + 1)) · f⁻²|⁻¹
= |2a · f¹/² · f⁻²|⁻¹ = |2a · f⁻³/²|⁻¹ = f³/²/(2a) = (1 + 4a²α²)³/²/(2a)

Versore normale:

n = ρ · (dt/dα) · (dα/ds)
= [(1 + 4a²α²)³/²/(2a)] · (-4a²α · f⁻³/², 2a · f⁻³/²) · f⁻¹/²
= (-2aα/√(1 + 4a²α²), 1/√(1 + 4a²α²))

Verifica: |n|² = 4a²α²/(1 + 4a²α²) + 1/(1 + 4a²α²) = (4a²α² + 1)/(1 + 4a²α²) = 1 ✓

Cinematica: Moto del Punto lungo una Curva

Una volta calcolato il triedro di Frenet, possiamo descrivere il moto di un punto che si muove lungo la curva.

Velocità

La velocità è sempre tangenziale alla curva:

v = ṡt = (ds/dt)t

dove ṡ = ds/dt è la velocità scalare (rapidità) del punto lungo la curva.

Per calcolare ṡ, usiamo la regola della catena:

ṡ = ds/dt = (ds/dα) · (dα/dt) = |dX/dα| · α̇

Accelerazione

L'accelerazione ha due componenti:

a = s̈t + (1/ρ)ṡ²n = s̈t + κṡ²n

dove:

• s̈t: accelerazione tangenziale (variazione della rapidità)

• (1/ρ)ṡ²n: accelerazione centripeta (cambio di direzione)

Per calcolare s̈:

s̈ = d²s/dt² = d/dt(ds/dt) = d/dt(|dX/dα| · α̇)

Esempio: Moto sulla Circonferenza

Consideriamo il moto su una circonferenza di raggio R con parametro θ(t):

X(θ) = (R cos θ, R sin θ)
t = (-sin θ, cos θ)
n = (-cos θ, -sin θ)
ρ = R

Velocità:

ṡ = (ds/dθ) · θ̇ = R · θ̇
v = Rθ̇(-sin θ, cos θ)

Accelerazione:

Calcoliamo s̈:

s̈ = d/dt(Rθ̇) = Rθ̈

Quindi:

a = Rθ̈(-sin θ, cos θ) + (1/R)(Rθ̇)²(-cos θ, -sin θ)
= Rθ̈(-sin θ, cos θ) + Rθ̇²(-cos θ, -sin θ)

Interpretazione: Il primo termine è l'accelerazione tangenziale (proporzionale a θ̈), il secondo è l'accelerazione centripeta (proporzionale a θ̇²).

Esempio: Moto sulla Parabola

Per la parabola X(α) = (α, aα²) con parametro α(t):

|dX/dα| = √(1 + 4a²α²)
ρ = (1 + 4a²α²)³/²/(2a)

Velocità:

ṡ = √(1 + 4a²α²) · α̇
v = (1, 2aα) · α̇

Accelerazione:

Calcoliamo s̈:

s̈ = d/dt[√(1 + 4a²α²) · α̇] = (1/2)√(1 + 4a²α²)⁻¹ · 8a²α · α̇² + √(1 + 4a²α²) · α̈
= (4a²α · α̇²)/√(1 + 4a²α²) + α̈√(1 + 4a²α²)

L'accelerazione completa è:

a = s̈t + (1/ρ)ṡ²n
= [(4a²α · α̇²)/√(1 + 4a²α²) + α̈√(1 + 4a²α²)] · (1/√(1 + 4a²α²), 2aα/√(1 + 4a²α²))
+ [2a/(1 + 4a²α²)³/²] · (1 + 4a²α²) · α̇² · (-2aα/√(1 + 4a²α²), 1/√(1 + 4a²α²))

Esempio 3: Moto Circolare con Velocità Angolare Variabile

Problema

Un punto si muove su una circonferenza di raggio R secondo la legge:

X(t) = (R cos[(a/2)t² + ωt], R sin[(a/2)t² + ωt])

dove a e ω sono costanti. Calcolare il triedro di Frenet e il moto del punto.

Soluzione

Poniamo per semplicità y = (a/2)t² + ωt. Allora X(t) = (R cos y, R sin y)

Passo 1: Versore Tangente

Calcoliamo dX/dt:

dX/dt = (-R sin y · ẏ, R cos y · ẏ) = (-R sin y · (at + ω), R cos y · (at + ω))

Ascissa curvilinea:

|dX/dt| = R|at + ω|√(sin²y + cos²y) = R(at + ω)

(assumendo at + ω > 0)

Quindi:

dt/ds = 1/[R(at + ω)]

Versore tangente:

t = (dX/dt) · (dt/ds) = (-R sin y · (at + ω), R cos y · (at + ω)) · 1/[R(at + ω)]
= (-sin y, cos y)

Passo 2: Versore Normale

Calcoliamo dt/dt:

dt/dt = (-cos y · ẏ, -sin y · ẏ) = (-cos y · (at + ω), -sin y · (at + ω))

Raggio di curvatura:

ρ = |dt/dt · dt/ds|⁻¹ = |(-cos y · (at + ω), -sin y · (at + ω)) · 1/[R(at + ω)]|⁻¹
= |√(cos²y + sin²y)/R|⁻¹ = R

Versore normale:

n = ρ · (dt/dt) · (dt/ds) = R · (-cos y · (at + ω), -sin y · (at + ω)) · 1/[R(at + ω)]
= (-cos y, -sin y)

Passo 3: Moto del Punto

Velocità: Abbiamo già ṡ = ds/dt = R(at + ω)

v = ṡt = R(at + ω)(-sin y, cos y)

Accelerazione: In questo caso abbiamo già ds/dt, quindi:

s̈ = d/dt[R(at + ω)] = Ra

L'accelerazione è:

a = s̈t + (1/ρ)ṡ²n
= Ra(-sin y, cos y) + (1/R)[R(at + ω)]²(-cos y, -sin y)
= Ra(-sin y, cos y) + R(at + ω)²(-cos y, -sin y)

Interpretazione: La componente tangenziale dell'accelerazione è costante (Ra), mentre quella centripeta aumenta quadraticamente con t.

Applicazioni del Triedro di Frenet

1. Robotica e Traiettorie

Nel controllo di robot e veicoli autonomi, il triedro di Frenet viene usato per:

• Pianificare traiettorie smooth con curvatura controllata

• Calcolare le forze necessarie per seguire un percorso

• Definire sistemi di coordinate mobili lungo il percorso

2. Grafica Computerizzata

Per animare oggetti lungo curve (camera path, etc.), il triedro fornisce un riferimento naturale per orientare l'oggetto lungo il percorso.
