# Dispense-Facolt-Ingegneria-Robotica-UniPa
Raccolta di dispense, appunti e materiali per il corso di Ingegneria Robotica (Università di Palermo). Il progetto organizza le materie per anno accademico e permette di aprire contenuti in formato HTML o PDF tramite un’interfaccia semplice. Alcune materie sono ancora in preparazione o non ancora sostenute.
